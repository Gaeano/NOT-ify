# 🎵 NOT-ify
### A Midterm Mobile Development Project 
NOT-ify is a mobile application designed to clone **Spotify** that allows users to explore the global database of music releases (powered by Discogs). It features a UI where users can search for artists, view detailed album credits, manage their digital library, and stream music by playing the integrated YouTube videos associated with each release.

## Features 
1. Database Search
2. Now Playing
3. Library
4. Detailed Song Details
5. Artist Discography Profiles

### Additional Features
6. **_Genre Randomizer_** - A feature that allows users to step outside their usual listening habits by generating a random musical genre and presenting a list of releases from that category. It mimics the experience of blindly shuffling through different sections of a record store bin to discover new music.
7. **_Upcoming Concerts & Ticket Prices_** - A live event tracking feature that displays upcoming tour dates, venue locations, and estimated ticket prices for the artists users are currently exploring, helping them easily plan their next concert experience.

## Technologies Used
- **Development Environment**: Android Studio
- **Frontend**: Native Android XML Layouts
- **Database**: Firebase Firestore
- **Authentication**: Firebase Authentication
- **API Integration**:
    1. Data Source: Discogs API v2
    2. Networking Client: Retrofit v2
    3. JSON parsing: Gson
- **Media Playback**: android-youtube-player library
- **Image Loading**: Glide
